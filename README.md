Kalkulator dibuat untuk menghitung 2 angka, dalam hal penjumlahan, pengurangan, perkalian, dan pembagian. Website ini dibuat dengan menggunakan **HTML (Hypertext Markup Language)**, **CSS (Cascading Style Sheet)** dan **PHP (Hypertext Preprocessor)**.

# Requirements
- XAMPP : PHP >= 7.4.11
- Google Chrome 
- Visual Studio Code

## Instalation
* Download [XAMPP](https://www.apachefriends.org/download.html) sesuai OS (Operating System) masing-masing, kemudian install sesuai petunjuk.
* Download [Visual Studio Code](https://code.visualstudio.com/download) sesuai OS (Operating System) masing-masing, kemudian install sesuai petunjuk.
  
  ## Usage
1. Letakan file php dan css ke dalam   
    > *C:\xampp\htdocs*  
2. Jalankan *XAMPP -> Apache*.
3. Buka Browser ketikan 
   > *localhost/kalkulator*.
